## üõ†Ô∏è Development Setup

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation
1. Clone the repository
```bash
git clone https://github.com/Simiyuoliver/vue-chat-app.git
cd vue-chat-app
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run development server
```bash

# or

```
